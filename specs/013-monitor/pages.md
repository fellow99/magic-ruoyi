# 系统监控模块前端页面文档（013-monitor/pages.md）

> magic-ruoyi 系统监控模块前端页面定义。
>
> 版本: 1.0.0 | 日期: 2026-04-29

---

## 1. 页面概览

| 页面 | 文件路径 | 路由 | 描述 |
|------|----------|------|------|
| 在线用户 | `src/views/monitor/online/index.vue` | `/monitor/online` | 在线用户管理 |
| 登录日志 | `src/views/monitor/logininfor/index.vue` | `/monitor/logininfor` | 登录日志查询 |
| 操作日志 | `src/views/monitor/operlog/index.vue` | `/monitor/operlog` | 操作日志查询 |
| 缓存监控 | `src/views/monitor/cache/index.vue` | `/monitor/cache` | Redis 缓存监控 |

**公共依赖**:
- `@/api/monitor/online` - 在线用户 API
- `@/api/monitor/loginInfo` - 登录日志 API
- `@/api/monitor/operlog` - 操作日志 API
- `@/api/monitor/cache` - 缓存监控 API
- `pagination` - 分页组件
- `right-toolbar` - 右侧工具栏

│                                                     │
