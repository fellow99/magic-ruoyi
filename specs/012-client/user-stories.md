## 1. 客户端查询场景

### US-001 查询客户端列表

**作为** 系统管理员
**我希望** 查看系统中所有 OAuth 客户端配置
**以便** 管理和监控客户端安全策略

**验收标准**:
- 支持按客户端 ID 搜索
- 支持按客户端 Key 搜索
- 支持按授权类型筛选
- 支持按设备类型筛选
- 支持按状态筛选
- 列表显示客户端 ID、Key、秘钥、授权类型、设备类型、Token 参数、状态

**优先级**: 必须

---

## 2. 客户端维护场景

### US-002 新增客户端

**作为** 系统管理员
**我希望** 添加新的 OAuth 客户端
**以便** 支持新的应用接入系统

**验收标准**:
- 表单包含客户端 Key、秘钥、授权类型、设备类型、Token 参数、状态
- 客户端 ID 自动生成（UUID）
- 客户端秘钥可自动生成或手动输入
- 授权类型支持多选
- Token 活跃超时和固定超时为必填项
- 提交成功后刷新列表

**优先级**: 必须

---

### US-003 修改客户端

**作为** 系统管理员
**我希望** 修改已有客户端的配置
**以便** 调整客户端安全策略

**验收标准**:
- 点击修改按钮弹出对话框
- 表单自动填充当前数据
- 客户端 ID 不可修改
- 提交成功后刷新列表

**优先级**: 必须

---

### US-004 删除客户端

**作为** 系统管理员
**我希望** 删除不再使用的客户端
**以便** 保持客户端配置整洁

**验收标准**:
- 支持单条和批量删除
- 删除前弹出确认对话框
- 删除成功后自动刷新列表

**优先级**: 必须

---

### US-005 切换客户端状态

**作为** 系统管理员
**我希望** 快速启用或停用客户端
**以便** 临时控制客户端访问权限

**验收标准**:
- 支持通过状态切换按钮快速修改
- 停用的客户端无法用于登录
- 状态修改后立即生效

**优先级**: 应该

---
