# 工作流模块前端页面文档（015-workflow/pages.md）

> magic-ruoyi 工作流模块前端页面定义。
>
> 版本: 1.0.0 | 日期: 2026-04-29

---

## 1. 页面概览

| 页面 | 文件路径 | 路由 | 描述 |
|------|----------|------|------|
| 流程分类 | `src/views/workflow/category/index.vue` | `/workflow/category` | 流程分类树形管理 |
| 流程定义 | `src/views/workflow/processDefinition/index.vue` | `/workflow/processDefinition` | 流程定义管理与发布 |
| 流程实例 | `src/views/workflow/processInstance/index.vue` | `/workflow/processInstance` | 流程实例管理 |
| 任务管理 | `src/views/workflow/task/index.vue` | `/workflow/task` | 待办/已办/抄送任务 |
| 请假流程 | `src/views/workflow/leave/index.vue` | `/workflow/leave` | 请假申请示例 |
| SpEL 表达式 | `src/views/workflow/spel/index.vue` | `/workflow/spel` | SpEL 表达式管理 |

**公共依赖**:
- `@/api/workflow/category` - 流程分类 API
- `@/api/workflow/definition` - 流程定义 API
- `@/api/workflow/instance` - 流程实例 API
- `@/api/workflow/task` - 任务 API
- `@/api/workflow/spel` - SpEL API
- `Process` - 流程组件
- `pagination` - 分页组件

## 3. 流程定义页面（processDefinition/index.vue）

### 3.1 页面布局

```
┌────────────────────────────────────────────────────┐
│  [搜索区域]                                         │
│  ┌──────────────────────────────────────────────┐  │
│  │ 流程编码  流程名称  分类  发布状态  [搜索]    │  │
│  └──────────────────────────────────────────────┘  │
│                                                     │
│  ┌──────────────────────────────────────────────┐  │
│  │ [新增][导入][发布][删除]                       │  │
│  │ 流程定义列表表格                              │  │
│  │ 编码|名称|版本|分类|发布状态|激活状态|时间     │  │
│  │ [分页控件]                                    │  │
│  └──────────────────────────────────────────────┘  │
└────────────────────────────────────────────────────┘
```

### 3.2 核心交互

#### 3.2.1 导入流程定义

```
handleImport()
  ├── 打开文件上传对话框
  ├── 上传 ZIP/XML 文件
  ├── 调用 importDef(data)
  └── 刷新列表
```

#### 3.2.2 发布/取消发布

```
handlePublish(row)
  ├── 若 isPublish=0 → publish(id)
  ├── 若 isPublish=1 → unPublish(id)
  └── 刷新列表
```

#### 3.2.3 激活/挂起

```
handleActive(row)
  ├── 调用 active(definitionId, !activityStatus)
  └── 刷新列表
```

## 5. 任务管理页面（task/index.vue）

### 5.1 页面布局

```
┌────────────────────────────────────────────────────┐
│  [Tab 切换] 待办 | 已办 | 抄送 | 全部待办 | 全部已办 │
│                                                     │
│  [搜索区域]                                         │
│  ┌──────────────────────────────────────────────┐  │
│  │ 流程编码  流程名称  节点名称  [搜索]           │  │
│  └──────────────────────────────────────────────┘  │
│                                                     │
│  ┌──────────────────────────────────────────────┐  │
│  │ 任务列表表格                                  │  │
│  │ 流程|节点|业务编码|业务标题|状态|时间          │  │
│  │ [分页控件]                                    │  │
│  └──────────────────────────────────────────────┘  │
└────────────────────────────────────────────────────┘
```

### 5.2 核心交互

#### 5.2.1 审批任务

```
handleComplete(row)
  ├── 打开审批对话框
  ├── 填写审批意见
  ├── 调用 completeTask(data)
  └── 刷新列表
```

#### 5.2.2 驳回任务

```
handleBack(row)
  ├── 获取可驳回节点 getBackTaskNode(taskId, nodeCode)
  ├── 选择驳回节点
  ├── 调用 backProcess(data)
  └── 刷新列表
```

#### 5.2.3 转办/委派

```
handleTransfer(row)
  ├── 选择目标用户
  ├── 调用 taskOperation(data, 'transferTask')
  └── 刷新列表
```

#### 5.2.4 催办任务

```
handleUrge(row)
  ├── 填写催办理由
  ├── 调用 urgeTask(data)
  └── 提示"催办成功"
```

## 7. SpEL 表达式页面（spel/index.vue）

### 7.1 页面布局

```
┌────────────────────────────────────────────────────┐
│  [搜索区域]                                         │
│  ┌──────────────────────────────────────────────┐  │
│  │ 组件名称  方法名  状态  [搜索][重置]           │  │
│  └──────────────────────────────────────────────┘  │
│                                                     │
│  ┌──────────────────────────────────────────────┐  │
│  │ [新增][修改][删除]                             │  │
│  │ SpEL 表达式列表表格                           │  │
│  │ 组件名|方法名|参数|预览SpEL|状态|备注          │  │
│  │ [分页控件]                                    │  │
│  └──────────────────────────────────────────────┘  │
└────────────────────────────────────────────────────┘
```

### 7.2 表单字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 组件名称 | 文本输入 | 是 | SpEL 组件名称 |
| 方法名 | 文本输入 | 是 | 方法名称 |
| 参数 | 文本输入 | 否 | 方法参数 |
| 预览 SpEL | 文本展示 | - | 自动生成的 SpEL 表达式 |
| 状态 | 单选按钮组 | 否 | 0=正常, 1=停用 |
| 备注 | 文本域 | 否 | 补充说明 |
