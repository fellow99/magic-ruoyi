## 2. 文件管理页面（index.vue）

### 2.1 页面布局

```
┌────────────────────────────────────────────────────┐
│  [搜索区域]                                         │
│  ┌──────────────────────────────────────────────┐  │
│  │ 文件名  原名  后缀  服务商  创建时间          │  │
│  │                              [搜索][重置]     │  │
│  └──────────────────────────────────────────────┘  │
│                                                     │
│  ┌──────────────────────────────────────────────┐  │
│  │ [删除]                                        │  │
│  │ ┌──────────────────────────────────────────┐ │  │
│  │ │ 文件列表表格                              │ │  │
│  │ │ 文件名 | 原名 | 后缀 | URL | 服务商 | 时间│ │  │
│  │ └──────────────────────────────────────────┘ │  │
│  │ [分页控件]                                    │  │
│  └──────────────────────────────────────────────┘  │
└────────────────────────────────────────────────────┘
```

### 2.2 组件结构

| 元素 | 类型 | 说明 |
|------|------|------|
| 文件名输入 | `<el-input>` | 模糊搜索 |
| 原名输入 | `<el-input>` | 模糊搜索 |
| 后缀输入 | `<el-input>` | 精确匹配 |
| 服务商输入 | `<el-input>` | 精确匹配 |
| 创建时间 | `<el-date-picker>` | 日期范围选择 |
| 删除按钮 | `<el-button>` | 权限: `system:oss:remove` |
| 数据表格 | `<el-table>` | 支持多选 |
| 文件 URL | `<a>` 链接 | 点击新窗口预览 |

### 2.3 表单数据模型

```typescript
interface OssVO extends BaseEntity {
  ossId: string | number;
  fileName: string;
  originalName: string;
  fileSuffix: string;
  url: string;
  createByName: string;
  service: string;
}

interface OssQuery extends PageQuery {
  fileName: string;
  originalName: string;
  fileSuffix: string;
  createTime: string;
  service: string;
  orderByColumn: string;
  isAsc: string;
}
```

### 2.4 核心交互逻辑

#### 2.4.1 删除操作

```
handleDelete(row?)
```

## 3. 配置管理页面（config.vue）

### 3.1 页面布局

```
┌────────────────────────────────────────────────────┐
│  ┌──────────────────────────────────────────────┐  │
│  │ [新增][修改][删除]                             │  │
│  │ ┌──────────────────────────────────────────┐ │  │
│  │ │ 配置列表表格                              │ │  │
│  │ │ 名称 | 端点 | 桶 | 服务商 | 默认 | 状态   │ │  │
│  │ └──────────────────────────────────────────┘ │  │
│  │ [分页控件]                                    │  │
│  └──────────────────────────────────────────────┘  │
└────────────────────────────────────────────────────┘
```

### 3.2 配置表单字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 配置名称 | 文本输入 | 是 | 配置显示名称 |
| 访问站点 | 文本输入 | 是 | OSS 端点 URL |
| 自定义域名 | 文本输入 | 否 | CDN 域名 |
| 存储桶 | 文本输入 | 是 | Bucket 名称 |
| 前缀 | 文本输入 | 否 | 文件路径前缀 |
| 访问密钥 | 文本输入 | 是 | Access Key |
| 秘钥 | 文本输入 | 是 | Secret Key |
| 是否 HTTPS | 单选 | 否 | Y/N |
| 域 | 文本输入 | 否 | Region |
| 服务商 | 下拉选择 | 是 | 存储服务商 |
| 是否默认 | 单选 | 否 | Y/N |
| 状态 | 单选 | 否 | 0=正常, 1=停用 |
| 访问策略 | 下拉选择 | 否 | 桶访问策略 |
