# 参数配置模块规格文档（009-config/spec.md）

> magic-ruoyi 参数配置模块。定义系统参数配置的增删改查、缓存刷新功能。
>
> 版本: 1.0.0 | 日期: 2026-04-29

---

## 1. 模块概述

参数配置模块负责维护系统运行时的动态参数，如系统名称、注册开关、验证码开关等。参数分为系统内置（Y）和自定义（N）两种类型，系统内置参数不可随意删除。

### 1.1 技术栈

| 组件 | 技术选型 |
|------|----------|
| 后端框架 | Spring Boot + MyBatis-Plus |
| 前端框架 | Vue 3.5 + Element Plus |
| 缓存 | Redis（参数缓存） |

### 1.2 核心功能

- 参数配置 CRUD
- 按参数键名查询参数值
- 按参数键名更新参数值
- 参数缓存刷新
- 参数数据导出

---

## 2. 功能规格

### 2.1 参数列表

**查询条件**:

| 条件 | 类型 | 说明 |
|------|------|------|
| 参数名称 | 文本输入 | 模糊匹配 configName |
| 参数键名 | 文本输入 | 模糊匹配 configKey |
| 系统内置 | 下拉选择 | Y=是, N=否（字典 sys_yes_no） |
| 创建时间 | 日期范围 | 时间范围筛选 |

**列表字段**:

| 字段 | 说明 |
|------|------|
| 参数名称 | configName |
| 参数键名 | configKey |
| 参数键值 | configValue |
| 系统内置 | configType（字典标签展示） |
| 备注 | remark |
| 创建时间 | createTime |

### 2.2 参数新增/修改

**表单字段**:

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| 参数名称 | 文本输入 | 是 | - | 参数显示名称 |
| 参数键名 | 文本输入 | 是 | - | 参数唯一标识（如 sys.register） |
| 参数键值 | 文本域 | 是 | - | 参数值 |
| 系统内置 | 单选按钮组 | 否 | Y | Y=是, N=否 |
| 备注 | 文本域 | 否 | - | 补充说明 |

**校验规则**:
- 参数名称不能为空
- 参数键名不能为空
- 参数键值不能为空

### 2.3 参数删除

- 支持单条删除和批量删除
- 删除前弹出确认对话框

### 2.4 刷新缓存

- 调用 `DELETE /system/config/refreshCache`
- 清除 Redis 中的参数缓存

## 3. 权限控制

| 权限标识 | 功能 | 按钮/操作 |
|----------|------|-----------|
| `system:config:list` | 查看参数列表 | 页面访问 |
| `system:config:query` | 查看参数详情 | 查看详情 |
| `system:config:add` | 新增参数 | 新增按钮 |
| `system:config:edit` | 修改参数 | 修改按钮 |
| `system:config:remove` | 删除参数 | 删除按钮、刷新缓存 |
| `system:config:export` | 导出参数 | 导出按钮 |

---

## 4. 数据模型

### 4.1 核心实体: SysConfig

| 字段 | 类型 | 说明 |
|------|------|------|
| configId | Long | 参数 ID（主键） |
| configName | String | 参数名称 |
| configKey | String | 参数键名 |
| configValue | String | 参数键值 |
| configType | String | 系统内置（Y=是, N=否） |
| remark | String | 备注 |
| createTime | Date | 创建时间 |

---

## 5. 使用的字典

| 字典类型 | 用途 | 枚举值 |
|----------|------|--------|
| sys_yes_no | 系统内置标识 | Y=是, N=否 |

## 6. 常见系统参数

| 参数键名 | 参数名称 | 默认值 | 说明 |
|----------|----------|--------|------|
| sys.register | 注册开关 | false | 是否允许用户注册 |
| sys.captcha.enabled | 验证码开关 | true | 是否启用验证码 |
| sys.account.captchaEnabled | 账户验证码开关 | true | 账户操作是否需要验证码 |
