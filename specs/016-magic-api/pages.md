## 2. Magic-API 编辑器

```

│  ┌──────────────────────────┐ │  ┌────────────────────────────┐ │
│  │ 接口树/函数树/数据源列表  │ │  │ 代码编辑器                  │ │
│  │                          │ │  │                             │ │
│  │ ├─ 用户管理              │ │  │ // 编写 MagicScript 脚本    │ │
│  │ │  ├─ 查询用户列表       │ │  │ var list = db.select(...)   │ │
│  │ │  └─ 新增用户           │ │  │ return list;                │ │
```

### 2.2 功能区域

#### 2.2.1 接口管理

| 功能 | 说明 |
|------|------|
| 接口树 | 按分组展示所有接口 |

| 编辑接口 | 编写 MagicScript 脚本 |
| 测试接口 | 内置调试工具发送请求 |
| 发布/下线 | 控制接口可用性 |
| 导入/导出 | 接口配置迁移 |

#### 2.2.2 函数管理

| 功能 | 说明 |
|------|------|
| 函数列表 | 展示所有自定义函数 |
| 新增函数 | 创建可复用函数 |
| 编辑函数 | 编写函数体脚本 |
| 测试函数 | 在线测试函数执行 |

#### 2.2.3 数据源管理

| 功能 | 说明 |
|------|------|
| 数据源列表 | 展示所有配置的数据源 |
| 新增数据源 | 配置数据库连接 |
| 编辑数据源 | 修改连接信息 |
| 测试连接 | 验证数据库连通性 |

#### 2.2.4 在线调试

| 功能 | 说明 |
|------|------|
| 请求参数 | 设置 Query/Body/Header 参数 |
| 发送请求 | 执行接口并获取响应 |
| 响应展示 | 显示响应头、响应体 |
| 日志展示 | 显示执行日志和性能数据 |

### 2.3 编辑器特性

## 3. 访问控制

### 3.1 编辑器访问

- 编辑器路径: `/magic/web`
- 通过 Sa-Token 进行访问控制
- 仅授权用户可访问编辑器

### 3.2 接口访问

- 动态接口路径: `/magic/api/{path}`
- 接口权限由编辑器中配置决定
- 可配置是否需要 Token 认证

---

## 4. 配置说明

### 4.1 application.yml 配置

```yaml
magic-api:
  web: /magic/web              # 编辑器访问路径
  prefix: /magic/api           # 接口执行前缀
  resource:
    datasource: magic-api      # 使用数据库存储配置
  auth:
    enabled: true              # 启用编辑器认证
```

### 4.2 存储方式

Magic-API 支持多种存储方式：

| 存储方式 | 说明 |
|----------|------|
| 数据库 | 推荐，支持多实例共享 |
| 文件 | 本地文件存储 |
| Redis | 缓存存储 |
