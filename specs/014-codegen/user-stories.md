# 代码生成模块用户故事（014-codegen/user-stories.md）

> magic-ruoyi 代码生成模块用户故事。描述开发者在代码生成场景中的需求。
>
> 版本: 1.0.0 | 日期: 2026-04-29

---

## 1. 表导入场景

### US-001 导入数据库表

**作为** 开发者
**我希望** 从数据库中选择表并导入到代码生成器
**以便** 基于已有表结构自动生成 CRUD 代码

**验收标准**:
- 打开导入表弹窗后显示数据库表列表
- 支持按数据源、表名称、表描述筛选
- 可多选表进行批量导入
- 导入成功后在主列表显示已导入的表
- 重复导入同一张表时给出提示

**优先级**: 必须

---

## 2. 配置编辑场景

### US-002 编辑表基本信息

**作为** 开发者
**我希望** 编辑生成表的基本信息（类名、包名、模块名、作者等）
**以便** 定制生成代码的命名和组织结构

**验收标准**:
- 可编辑表描述、类名、作者
- 可设置包名、模块名、业务名、功能名
- 可选择模板分类（单表/树表/主子表）
- 树表场景下可配置树编码、树父编码、树名称字段
- 可配置上级菜单用于生成菜单 SQL

**优先级**: 必须

### US-003 编辑字段信息

**作为** 开发者
**我希望** 编辑每个字段的生成配置
**以便** 控制字段在代码中的表现

**验收标准**:
- 可编辑字段描述
- 可选择 Java 类型（Long/String/Integer/Double/BigDecimal/Date/Boolean）
- 可编辑 Java 属性名
- 可勾选字段是否参与新增、编辑、列表、查询
- 可选择查询方式（EQ/NE/GT/GE/LT/LE/LIKE/BETWEEN）
- 可勾选字段是否必填
- 可选择 HTML 组件类型（文本框/下拉框/日期控件/图片上传等）
- 可绑定字典类型用于下拉框/单选框/复选框

**优先级**: 必须

### US-004 编辑生成信息

**作为** 开发者
**我希望** 配置代码生成方式和路径
**以便** 控制代码输出位置

**验收标准**:
- 可选择生成方式：下载 ZIP 或生成到自定义路径
- 选择自定义路径时可输入目标目录
- 可配置上级菜单用于生成菜单 SQL 脚本

**优先级**: 必须

---

## 3. 代码预览与生成场景

### US-005 预览生成代码

**作为** 开发者
**我希望** 在线预览生成的代码文件
**以便** 在正式生成前检查代码质量

**验收标准**:
- 点击预览按钮后弹出代码预览对话框
- 以 Tab 页签形式展示多个文件（domain.java、mapper.java、service.java 等）
- 每个文件显示语法高亮的代码内容
- 支持一键复制代码到剪贴板
- 页签名称显示文件名（去除 .vm 后缀）

**优先级**: 必须

### US-006 下载生成代码

**作为** 开发者
**我希望** 将生成的代码打包为 ZIP 文件下载
**以便** 在本地项目中解压使用

**验收标准**:
- 支持单表和多表批量下载
- 下载文件名为 `ruoyi.zip`
- ZIP 中包含完整的后端和前端代码文件
- 目录结构按模块和包名组织

**优先级**: 必须

### US-007 生成代码到自定义路径

**作为** 开发者
**我希望** 将代码直接生成到服务器指定目录
**以便** 无需手动复制文件

**验收标准**:
- 仅当 genType=1 时生效
- 生成成功后提示目标路径
- 文件按包名和模块名自动创建目录结构

**优先级**: 应该

---

## 4. 表管理场景

### US-008 同步数据库表结构

**作为** 开发者
**我希望** 同步已导入表的数据库结构变更
**以便** 当数据库表字段变更后更新生成配置

**验收标准**:
- 点击同步按钮后弹出确认对话框
- 确认后从数据库重新读取表结构
- 更新 gen_table_column 中的字段信息
- 同步成功后提示成功

**优先级**: 应该

### US-009 删除生成配置

**作为** 开发者
**我希望** 删除不再需要的生成配置
**以便** 保持代码生成列表整洁

**验收标准**:
- 支持单条和批量删除
- 删除前弹出确认对话框
- 删除后从 gen_table 和 gen_table_column 中移除记录
- 不影响原始数据库表

**优先级**: 必须

---

## 5. 多数据源场景

### US-010 切换数据源

**作为** 开发者
**我希望** 在多个数据源之间切换
**以便** 从不同数据库读取表结构

**验收标准**:
- 列表页可按数据源筛选
- 导入表弹窗可选择数据源
- 数据源下拉列表显示所有可用数据源名称

**优先级**: 应该
