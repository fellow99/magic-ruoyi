# 字典管理模块规格文档（008-dict/spec.md）

> magic-ruoyi 字典管理模块。定义字典类型和字典数据的增删改查、缓存刷新功能。
>
> 版本: 1.0.0 | 日期: 2026-04-29

---

## 1. 模块概述

字典管理模块负责维护系统中的字典数据，包括字典类型管理和字典数据管理两个子模块。字典用于统一管理系统中的枚举值、状态码、选项列表等静态数据，支持缓存机制提升查询性能。

### 1.1 技术栈

| 组件 | 技术选型 |
|------|----------|
| 后端框架 | Spring Boot + MyBatis-Plus |
| 前端框架 | Vue 3.5 + Element Plus |
| 缓存 | Redis（字典缓存） |
| 状态管理 | Pinia（dictStore） |

### 1.2 核心功能

- 字典类型 CRUD（增删改查）
- 字典数据 CRUD（按字典类型管理）
- 字典缓存刷新
- 字典数据导出
- 按字典类型查询字典数据（供前端动态加载）

### 1.3 模块结构

```
字典管理
├── 字典类型管理 (index.vue)     → /system/dict
└── 字典数据管理 (data.vue)      → /system/dict-data/index/:dictId
```

## 2. 字典类型管理

### 2.1 字典类型列表

**查询条件**:

| 条件 | 类型 | 说明 |
|------|------|------|
| 字典名称 | 文本输入 | 模糊匹配 dictName |
| 字典类型 | 文本输入 | 模糊匹配 dictType |
| 创建时间 | 日期范围 | 时间范围筛选 |

**列表字段**:

| 字段 | 说明 |
|------|------|
| 字典名称 | dictName |
| 字典类型 | dictType（可点击跳转到字典数据页） |
| 备注 | remark |
| 创建时间 | createTime |

### 2.2 字典类型新增/修改

**表单字段**:

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 字典名称 | 文本输入 | 是 | 字典显示名称 |
| 字典类型 | 文本输入 | 是 | 字典唯一标识（如 sys_normal_disable） |
| 备注 | 文本域 | 否 | 补充说明 |

**校验规则**:
- 字典名称不能为空
- 字典类型不能为空

### 2.3 字典类型删除

- 支持单条删除和批量删除
- 删除前弹出确认对话框
- 删除字典类型时级联删除关联的字典数据

### 2.4 刷新缓存

- 调用 `DELETE /system/dict/type/refreshCache`
- 清除 Redis 中的字典缓存
- 清除前端 dictStore 中的缓存

## 3. 字典数据管理

### 3.1 字典数据列表

**查询条件**:

| 条件 | 类型 | 说明 |
|------|------|------|
| 字典类型 | 下拉选择 | 选择字典类型 |
| 字典标签 | 文本输入 | 模糊匹配 dictLabel |

**列表字段**:

| 字段 | 说明 |
|------|------|
| 字典标签 | dictLabel（带样式标签展示） |
| 字典键值 | dictValue |
| 字典排序 | dictSort |
| 备注 | remark |
| 创建时间 | createTime |

### 3.2 字典数据新增/修改

**表单字段**:

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| 字典类型 | 文本输入（禁用） | - | 自动填充 | 所属字典类型 |
| 数据标签 | 文本输入 | 是 | - | 显示文本 |
| 数据键值 | 文本输入 | 是 | - | 存储值 |
| 样式属性 | 文本输入 | 否 | - | CSS 类名 |
| 显示排序 | 数字输入 | 是 | 0 | 排序值 |
| 回显样式 | 下拉选择 | 否 | primary | Tag 颜色类型 |
| 备注 | 文本域 | 否 | - | 补充说明 |

**回显样式选项**:

| 值 | 标签 |
|----|------|
| default | 默认 |
| primary | 主要 |
| success | 成功 |
| info | 信息 |
| warning | 警告 |
| danger | 危险 |

**校验规则**:
- 数据标签不能为空
- 数据键值不能为空
- 显示排序不能为空

### 3.3 字典数据删除

- 支持单条删除和批量删除
- 删除后清除对应字典类型的缓存

| 权限标识 | 功能 | 按钮/操作 |
|----------|------|-----------|
| `system:dict:list` | 查看字典列表 | 页面访问 |

## 5. 数据模型

### 5.1 核心实体: SysDictType

| 字段 | 类型 | 说明 |
|------|------|------|
| dictId | Long | 字典 ID（主键） |
| dictName | String | 字典名称 |
| dictType | String | 字典类型（唯一标识） |
| remark | String | 备注 |
| createTime | Date | 创建时间 |

### 5.2 核心实体: SysDictData

| 字段 | 类型 | 说明 |
|------|------|------|
| dictCode | Long | 字典数据 ID（主键） |
| dictType | String | 字典类型（关联 SysDictType） |
| dictLabel | String | 字典标签（显示文本） |
| dictValue | String | 字典键值（存储值） |
| cssClass | String | 样式属性（CSS 类名） |
| listClass | String | 回显样式（Tag 类型） |
| dictSort | Integer | 排序 |
| remark | String | 备注 |
| createTime | Date | 创建时间 |

### 5.3 关联关系

- SysDictType 1-N SysDictData: 一个字典类型包含多个字典数据

---

## 6. 缓存机制

- 字典数据缓存在 Redis 中
- Key 格式: `dict:{dictType}`
- 刷新缓存时清除所有字典缓存
- 前端 dictStore 同步管理本地字典缓存
