# 角色管理前端页面文档 (pages.md)

> magic-ruoyi 角色管理模块的前端页面定义。
>
> 版本: 1.0.0 | 最后更新: 2026-04-29

---

## 1. 页面清单

| 页面文件 | 路由路径 | 页面名称 | 功能说明 |
|----------|----------|----------|----------|
| `views/system/role/index.vue` | `/system/role` | 角色管理 | 角色列表、CRUD、菜单权限分配、数据权限配置 |
| `views/system/role/authUser.vue` | `/system/role-auth/user/:roleId` | 角色分配用户 | 已分配用户列表、添加/取消用户授权 |
| `views/system/role/selectUser.vue` | 弹窗组件 | 选择用户 | 未分配用户选择弹窗 |

## 2. 角色管理页面 (index.vue)

### 2.1 页面布局

```
+----------------------------------------------------------+
| 搜索区域（可折叠）                                          |
| [角色名称] [权限字符] [状态] [创建时间范围] [搜索] [重置]    |
+----------------------------------------------------------+
| 工具栏                                                    |
| [新增] [修改] [删除] [导出]              [显示/隐藏搜索]    |
+----------------------------------------------------------+
| 数据表格                                                  |
| [ ] | 角色名称 | 权限字符 | 顺序 | 状态 | 创建时间 | 操作   |
| [ ] | 超级管理员 | superadmin | 1 | [开关] | 2026-xx-xx | 修改 删除 数据权限 分配用户 |
| [ ] | 管理员 | admin | 3 | [开关] | 2026-xx-xx | 修改 删除 数据权限 分配用户 |
+----------------------------------------------------------+
| 分页组件                                                  |
| 共 xx 条  [上一页] 1 2 3 [下一页]  10条/页                 |
+----------------------------------------------------------+
```

### 2.2 搜索区域

| 字段 | 组件 | 说明 |
|------|------|------|
| 角色名称 | el-input | 模糊搜索，支持回车触发 |
| 权限字符 | el-input | 模糊搜索，支持回车触发 |
| 状态 | el-select | 下拉选择，选项来自字典 sys_normal_disable |
| 创建时间 | el-date-picker | 日期范围选择器，格式 YYYY-MM-DD HH:mm:ss |

### 2.3 工具栏按钮

| 按钮 | 权限标识 | 禁用条件 | 操作 |
|------|----------|----------|------|
| 新增 | system:role:add | 无 | 打开新增角色弹窗 |
| 修改 | system:role:edit | 未选中或选中多行时禁用 | 打开修改角色弹窗 |
| 删除 | system:role:remove | 未选中任何行时禁用 | 批量删除确认并执行 |
| 导出 | system:role:export | 无 | 导出当前查询结果为Excel |

### 2.4 数据表格

| 列名 | 字段 | 宽度 | 说明 |
|------|------|------|------|
| 选择框 | selection | 55px | 多选框 |
| 角色编号 | roleId | 120px | 默认隐藏（v-if="false"） |
| 角色名称 | roleName | 150px | 超出显示省略号 |
| 权限字符 | roleKey | 200px | 超出显示省略号 |
| 显示顺序 | roleSort | 100px | 数字 |
| 状态 | status | 100px | el-switch 开关组件，active-value="0"，inactive-value="1" |
| 创建时间 | createTime | - | 格式化显示 |
| 操作 | - | 180px | 固定右侧 |

### 2.5 操作列按钮

| 按钮 | 权限标识 | 显示条件 | 操作 |
|------|----------|----------|------|
| 修改 | system:role:edit | roleId !== 1 | 打开修改弹窗 |
| 删除 | system:role:remove | roleId !== 1 | 删除确认并执行 |
| 数据权限 | system:role:edit | roleId !== 1 | 打开数据权限配置弹窗 |
| 分配用户 | system:role:edit | roleId !== 1 | 跳转到用户分配页面 |

### 2.6 新增/修改角色弹窗

**弹窗标题**: "添加角色" 或 "修改角色"
**弹窗宽度**: 500px

**表单字段**:

| 字段 | 组件 | 必填 | 校验规则 | 说明 |
|------|------|------|----------|------|
| 角色名称 | el-input | 是 | 不能为空 | - |
| 权限字符 | el-input | 是 | 不能为空 | 带tooltip提示：控制器中定义的权限字符，如：@SaCheckRole('admin') |
| 角色顺序 | el-input-number | 是 | 不能为空 | 最小值0，右侧控制按钮 |
| 状态 | el-radio-group | 是 | - | 选项来自字典 sys_normal_disable |
| 菜单权限 | el-tree | 否 | - | 树形结构，支持勾选 |
| 备注 | el-input textarea | 否 | - | 多行文本 |

**菜单权限树操作**:

| 操作 | 组件 | 说明 |
|------|------|------|
| 展开/折叠 | el-checkbox | 切换所有节点的展开状态 |
| 全选/全不选 | el-checkbox | 切换所有节点的勾选状态 |
| 父子联动 | el-checkbox | 控制 check-strictly 属性 |

**菜单树配置**:

```
node-key: "id"
show-checkbox: true
check-strictly: !form.menuCheckStrictly
props: { label: 'label', children: 'children' }
```

### 2.7 数据权限配置弹窗

**弹窗标题**: "分配数据权限"
**弹窗宽度**: 500px

**表单字段**:

| 字段 | 组件 | 可编辑 | 说明 |
|------|------|--------|------|
| 角色名称 | el-input | 否（disabled） | 只读显示 |
| 权限字符 | el-input | 否（disabled） | 只读显示 |
| 权限范围 | el-select | 是 | 6个选项 |
| 数据权限 | el-tree | 条件显示 | 仅 dataScope="2" 时显示 |

**权限范围选项**:

| 值 | 标签 |
|----|------|
| 1 | 全部数据权限 |
| 2 | 自定数据权限 |
| 3 | 本部门数据权限 |
| 4 | 本部门及以下数据权限 |
| 5 | 仅本人数据权限 |
| 6 | 部门及以下或本人数据权限 |

**部门树操作**（仅自定义模式）:

| 操作 | 组件 | 说明 |
|------|------|------|
| 展开/折叠 | el-checkbox | 切换所有节点展开状态 |
| 全选/全不选 | el-checkbox | 切换所有节点勾选状态 |
| 父子联动 | el-checkbox | 控制 check-strictly 属性 |

**部门树配置**:

```
node-key: "id"
show-checkbox: true
default-expand-all: true
check-strictly: !form.deptCheckStrictly
props: { label: 'label', children: 'children' }
```

**联动行为**: 切换权限范围为非自定义时，自动清空部门树的勾选状态。

### 2.8 状态管理

**交互流程**:

1. 用户点击表格中的状态开关
2. 弹出确认对话框："确认要'启用/停用''角色名称'角色吗?"
3. 用户确认后调用 `changeRoleStatus` 接口
4. 成功则提示"启用/停用成功"
5. 失败则回滚开关状态

### 2.9 表单初始化值

```typescript
const initForm: RoleForm = {
  roleId: undefined,
  roleSort: 1,
  status: '0',
  roleName: '',
  roleKey: '',
  menuCheckStrictly: true,
  deptCheckStrictly: true,
  remark: '',
  dataScope: '1',
  menuIds: [],
  deptIds: []
};
```

### 2.10 查询参数

```typescript
queryParams: {
  pageNum: 1,
  pageSize: 10,
  roleName: '',
  roleKey: '',
  status: ''
}
```

## 3. 角色分配用户页面 (authUser.vue)

### 3.1 页面布局

```
+----------------------------------------------------------+
| 搜索区域（可折叠）                                          |
| [用户名称] [手机号码] [搜索] [重置]                         |
+----------------------------------------------------------+
| 工具栏                                                    |
| [添加用户] [批量取消授权] [关闭]          [显示/隐藏搜索]    |
+----------------------------------------------------------+
| 数据表格                                                  |
| [ ] | 用户名称 | 用户昵称 | 邮箱 | 手机 | 状态 | 创建时间 | 操作 |
| [ ] | admin | 管理员 | - | - | 正常 | 2026-xx-xx | 取消授权 |
+----------------------------------------------------------+
| 分页组件                                                  |
+----------------------------------------------------------+
| [选择用户弹窗组件]                                         |
+----------------------------------------------------------+
```

### 3.2 搜索区域

| 字段 | 组件 | 说明 |
|------|------|------|
| 用户名称 | el-input | 模糊搜索，支持回车触发 |
| 手机号码 | el-input | 模糊搜索，支持回车触发 |

### 3.3 工具栏按钮

| 按钮 | 权限标识 | 禁用条件 | 操作 |
|------|----------|----------|------|
| 添加用户 | system:role:add | 无 | 打开选择用户弹窗 |
| 批量取消授权 | system:role:remove | 未选中任何行时禁用 | 批量取消确认并执行 |
| 关闭 | 无 | 无 | 返回角色列表页面 |

### 3.4 数据表格

| 列名 | 字段 | 宽度 | 说明 |
|------|------|------|------|
| 选择框 | selection | 55px | 多选框 |
| 用户名称 | userName | - | 超出显示省略号 |
| 用户昵称 | nickName | - | 超出显示省略号 |
| 邮箱 | email | - | 超出显示省略号 |
| 手机 | phonenumber | - | 超出显示省略号 |
| 状态 | status | - | dict-tag 组件显示 |
| 创建时间 | createTime | 180px | 直接显示 |
| 操作 | - | - | 固定右侧 |

### 3.5 操作列

| 按钮 | 权限标识 | 操作 |
|------|----------|------|
| 取消授权 | system:role:remove | 取消该用户的角色授权 |

### 3.6 路由参数

| 参数 | 类型 | 说明 |
|------|------|------|
| roleId | String | 从路由参数 `route.params.roleId` 获取 |

### 3.7 查询参数

```typescript
queryParams: {
  pageNum: 1,
  pageSize: 10,
  roleId: route.params.roleId,
  userName: undefined,
  phonenumber: undefined
}
```

## 4. 选择用户弹窗 (selectUser.vue)

### 4.1 弹窗配置

| 属性 | 值 |
|------|-----|
| 标题 | 选择用户 |
| 宽度 | 800px |
| 位置 | top="5vh" |

### 4.2 Props

| 属性 | 类型 | 必填 | 说明 |
|------|------|------|------|
| roleId | Number/String | 是 | 角色ID |

### 4.3 搜索区域

| 字段 | 组件 | 说明 |
|------|------|------|
| 用户名称 | el-input | 模糊搜索 |
| 手机号码 | el-input | 模糊搜索 |

### 4.4 数据表格

| 列名 | 字段 | 宽度 | 说明 |
|------|------|------|------|
| 选择框 | selection | 55px | 多选框 |
| 用户名称 | userName | - | - |
| 用户昵称 | nickName | - | - |
| 邮箱 | email | - | - |
| 手机 | phonenumber | - | - |
| 状态 | status | - | dict-tag 显示 |
| 创建时间 | createTime | 180px | 格式化显示 |

**表格高度**: 260px

### 4.5 交互行为

| 行为 | 说明 |
|------|------|
| 点击行 | 切换该行的选中状态（Element Plus 的已知行为修复） |
| 多选 | 收集选中用户的 userId 列表 |
| 确定 | 调用批量授权接口，成功后触发父组件的 ok 事件并关闭弹窗 |
| 取消 | 关闭弹窗 |

### 4.6 暴露方法

| 方法 | 说明 |
|------|------|
| `show()` | 显示弹窗并加载未分配用户列表 |

### 4.7 事件

| 事件 | 参数 | 触发时机 |
|------|------|----------|
| ok | 无 | 用户选择并授权成功后 |
