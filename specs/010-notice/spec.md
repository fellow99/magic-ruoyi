# 通知公告模块规格文档（010-notice/spec.md）

> magic-ruoyi 通知公告模块。定义系统公告的发布、编辑、删除功能。
>
> 版本: 1.0.0 | 日期: 2026-04-29

---

## 1. 模块概述

通知公告模块负责管理系统内部公告的发布与维护，支持富文本编辑、公告类型分类、状态管理等功能。公告由管理员创建，面向系统用户展示。

### 1.1 技术栈

| 组件 | 技术选型 |
|------|----------|
| 后端框架 | Spring Boot + MyBatis-Plus |
| 前端框架 | Vue 3.5 + Element Plus |
| 富文本编辑器 | Editor 组件 |

### 1.2 核心功能

- 公告 CRUD（增删改查）
- 富文本内容编辑
- 公告类型分类（通知/公告）
- 公告状态管理（正常/关闭）
- 按创建者筛选

---

## 2. 功能规格

### 2.1 公告列表

**查询条件**:

| 条件 | 类型 | 说明 |
|------|------|------|
| 公告标题 | 文本输入 | 模糊匹配 noticeTitle |
| 操作人员 | 文本输入 | 按创建者名称筛选 |
| 类型 | 下拉选择 | 通知/公告（字典 sys_notice_type） |

**列表字段**:

| 字段 | 说明 |
|------|------|
| 公告标题 | noticeTitle |
| 公告类型 | noticeType（字典标签展示） |
| 状态 | status（字典标签展示） |
| 创建者 | createByName |
| 创建时间 | createTime |

### 2.2 公告新增/修改

**表单字段**:

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| 公告标题 | 文本输入 | 是 | - | 公告标题 |
| 公告类型 | 下拉选择 | 是 | - | 通知/公告 |
| 状态 | 单选按钮组 | 否 | 0（正常） | 正常/关闭 |
| 内容 | 富文本编辑器 | 否 | - | 公告正文 |

**校验规则**:
- 公告标题不能为空
- 公告类型不能为空

### 2.3 公告删除

- 支持单条删除和批量删除
- 删除前弹出确认对话框

---

## 3. 权限控制

| 权限标识 | 功能 | 按钮/操作 |
|----------|------|-----------|
| `system:notice:list` | 查看公告列表 | 页面访问 |
| `system:notice:query` | 查看公告详情 | 查看详情 |
| `system:notice:add` | 新增公告 | 新增按钮 |
| `system:notice:edit` | 修改公告 | 修改按钮 |
| `system:notice:remove` | 删除公告 | 删除按钮 |

---

## 4. 数据模型

### 4.1 核心实体: SysNotice

| 字段 | 类型 | 说明 |
|------|------|------|
| noticeId | Long | 公告 ID（主键） |
| noticeTitle | String | 公告标题 |
| noticeType | String | 公告类型（1=通知, 2=公告） |
| noticeContent | String | 公告内容（富文本 HTML） |
| status | String | 状态（0=正常, 1=关闭） |
| createByName | String | 创建者名称（关联查询） |
| remark | String | 备注 |
| createTime | Date | 创建时间 |

## 5. 使用的字典

| 字典类型 | 用途 | 枚举值 |
|----------|------|--------|
| sys_notice_type | 公告类型 | 1=通知, 2=公告 |
| sys_notice_status | 公告状态 | 0=正常, 1=关闭 |
